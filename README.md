# Fraud Detection for E-commerce and Bank Transactions

## Overview
A machine learning system to detect fraud in e-commerce and bank credit card transactions for Adey Innovations Inc.

## Datasets
- Fraud_Data.csv - E-commerce transactions
- IpAddress_to_Country.csv - IP to country mapping
- creditcard.csv - Bank credit card transactions

## Setup
```bash
pip install -r requirements.txt
```

## Project Structure
- notebooks/ - EDA and modeling notebooks
- src/ - Source code
- data/ - Raw and processed data (gitignored)
- models/ - Saved model artifacts
- tests/ - Unit tests
